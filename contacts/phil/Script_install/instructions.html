[edit][hide] What's Greasemonkey?

Greasemonkey "allows you to customize the way a web page displays or behaves, by using small bits of JavaScript." It is a Firefox extension that enables people to change web pages that they don't control.

Like anything else you load on your system, Greasemonkey can be used against you. Don't load scripts onto your system unless you trust the source.

WARNING:
  • Greasemonkey and Greasemonkey scripts are not officially supported by Amazon. The Global Helpdesk and the IT Support teams will only provide support on a best-effort basis
  • Greasemonkey scripts must not be used on sites that handle Restricted or Critical data

[edit][hide] How do I get them?

This page will guide you through the install of Greasemonkey and the scripts.

You will have to add Greasemonkey to Firefox, then restart the browser. Except for thin clients, your tabs will come back after the restart, so you can do this after you are set up.

Next, you have to click the "Install the scripts" button, and wait for all twelve install dialogs to pop up. Then, install all twelve by pressing the install buttons.

You can then close the install page and start exploring the scripts.

[edit][hide] What if I run into problems?

Again, these scripts are not officially supported by Amazon. The best you can do is check for updates, and if the problem isn't already fixed, reach out to jffrenc@.

[edit][hide] Disable Greasemonkey or individual scripts

To disable Greasemonkey entirely, you can click the monkey in the top right of your browser. It's in color when enabled, and black and white when disabled.

To disable individual scripts, click the down arrow to the right of the monkey, and uncheck the script you want to disable. This list only includes scripts for the page you are currently viewing. To see all of them, click the down arrow next to the monkey and click Manage User Scripts. You can disable or remove individual scripts using the buttons in each row.
